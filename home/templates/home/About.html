BlogLikho.com

BlogLikho is one of fully responsive website and advanced integrated editor which allows user to perform various operations while writing blogs, this editor helps user to make his/her blog intresting and well presentable. We also added the feature of sharing the blogs, commenting on blog and liking the blog, which make this more interactive. This website also has a functionality to get the most popular blog, recent blogs as well. We consider this website as one of our major project which has basic functions with both Front End and Back End functioning properly. For now we are keeping the website as simple as possible to use and with time we will go on adding new features which will assits the users
